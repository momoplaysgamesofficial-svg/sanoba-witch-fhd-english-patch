Sanoba Witch FHD English Patch

Patches the FHD version of Sabbat of the Witch with English text and UI.

Download Patch

Features

  • Replaces CN with NekoNyan’s official English translation
  • Replaces CN UI assets with English UI
  • Fixes possible word cutoff and word wrapping issues for English text

How to Install

Auto Installer

  1. Download and run sw_en_patcher.exe
  2. Confirm the detected Sanoba Witch game folder or browse to it manually
  3. Click Install

Manual Install

  1. Download and extract the patch_manual ZIP
  2. Locate the game folder
  3. Copy unencrypted.xp3 and version.dll into the game folder

Once installed, use the language option at the top right to switch the game to English.

Uninstall

  1. Remove unencrypted.xp3 and version.dll from the game folder
